# Glossary

- **DEX** — decentralized exchange.
- **Spot / Perpetuals** — immediate asset trading vs. derivative contracts with leverage.
- **Grid bot** — an automated strategy that places staggered buy/sell limit orders across a price range to profit from oscillation. See [Grid Trading](Grid-Trading.md).
- **Order book** — the set of resting buy/sell orders the matching engine pairs up.
- **Maker / Taker** — a maker adds liquidity (resting order); a taker removes it (crossing order).
- **STP (Self-Trade Prevention)** — an engine safety feature that stops a single account from trading against itself. See [Self-Trade Prevention](Self-Trade-Prevention.md).
- **Deposit vault** — the on-chain contract users deposit into; the exchange credits the corresponding trading balance. See [Deposits & Withdrawals](Deposits-and-Withdrawals.md).
- **SIWE** — Sign-In With Ethereum; wallet-based authentication.
- **Realized PnL** — profit/loss locked in from completed buy→sell cycles.
